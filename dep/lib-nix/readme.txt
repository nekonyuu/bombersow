Pour compiler le projet sous GNU/Linux & Unix, vous devez décompresser l'archive SFML et CSFML et avec un terminal :

cd SFML
su root
make install
cd ../CSFML
make install

Ensuite vous devez installer les paquets d'openal-dev et libsndfile, puis avec CodeBlocks, 
compiler en Release Linux ou Debug Linux, et lancez l'executable ;)
