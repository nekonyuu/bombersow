Pour compiler le projet sous GNU/Linux & Unix, vous devez installer les paquets d'openal-dev, libsndfile, et libglew
puis décompresser l'archive SFML et CSFML et avec un terminal :

cd SFML
su root
make install
cd ../CSFML
make install

Ensuite avec CodeBlocks, compiler en Release Linux ou Debug Linux, et lancez l'executable ;)
